# Vibes

`Text` supports every `vibe`.

```tsx
<Text vibe="positive">Post was successfully created!</Text>
```

## Examples

<Canvas>
  <div style={{ display: "flex", flexDirection: "column", gap: "12px" }}>
    <Text>This is the default vibe.</Text>
    <Text vibe="subdued">This text isn't as important.</Text>
    <Text vibe="positive">Post was successfully created!</Text>
    <Text vibe="warning">Your data usage is close to your limit</Text>
    <Text vibe="critical">Are you sure you want to delete this post?</Text>
    <Text vibe="attract">Click here to move forward</Text>
  </div>
</Canvas>
